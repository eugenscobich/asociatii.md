A new version of PMT is available!

What's new in this release (v2.12.0) ?

  • A new feature available for any employee allowing to suggest assessors for performance evaluation.

    Under "My Performance" screen, the employees can add assessors (from Endava or outside) to suggest to their people manager for performance evaluation. People Managers may use the suggested assessors by adding them in the "Performance Evaluation" screen.

  • People Managers can specify "Due date" when soliciting feedback.

    Now when sending assessment request for a person People Manager may add the "Due date" against each feedback form. The due date will appear in the notification message if specified otherwise a graceful request is added in the message.

  • Enhancements of the Feedback Notes screen.

    The following enhancements where applied on the Feedback Notes screen:

    - Ability to provide feedback against any skill not necessarily from the list of job role.

    - Ability to add feedback for another skill straight away after adding one feedback - no need to go back to the list of feedbacks and add from there, etc.

  • Optimized and improved performance of the following screens:

    - My People - performance improvement especially for huge number of employees;

    - Staff Objectives - view direct only (default) or indirect staff by triggering a checkbox;

    - Performance Evaluation - view direct only (default) or indirect staff by triggering a checkbox;

    - View Staff Performance.

  • And more enhancements and improvements as follows

    - Added the category filter under "My objectives", so now you can view the objectives by their category.

    - Employees or people managers are CCed (carbon copy) when notification messages are being sent.

    - Some statistics numbers with the progresses were added under the "Staff Objectives" and "Performance Evaluation" pages.
